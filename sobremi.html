Sobre mi

Soy diseñadora UX/UI y desarrolladora front-end. Me meto en todo el proceso, desde entender qué necesita el usuario hasta escribir el código que lo hace real, porque si no controlo cada parte me quedo con el gusanillo.

Uso Figma para diseñar y HTML, CSS y JavaScript para construir. Soy muy detallista, de las que ajustan un píxel a las 2 de la mañana y no se arrepienten.

Ahora mismo estoy aprendiendo Three.js porque la web convencional se me quedaba pequeña y necesitaba algo que me volviera a poner nerviosa. La IA también me tiene muy enganchada, no como buzzword, sino porque cambia de verdad cómo se diseñan las experiencias digitales.
